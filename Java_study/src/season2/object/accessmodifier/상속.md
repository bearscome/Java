|           | 같은 클래스 | 같은 패키지 | 클래스 상속<br/>(서브 클래스) | 관계 없음 |
|-----------|--------|--------|---------------------|-------| 
| public    | O      | O      | O                   | O     | 
| protected | O      | O      | O                   | X     |
| default   | O      | O      | X                   | X     |
| private   | O      | X      | X                   | X     |


private: 해당 클래스 내부에서만 접근 가능 <br/>
protected: 해당 클래스 및 하위 클래스에서 접근 가능 <br/>
default: 같은 패키지에서 접근 가능 <br/>
public: 모두 접근 가능
